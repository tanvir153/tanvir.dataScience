1. Introduction to Python

Python is a versatile, high-level, interpreted programming language known for its readability and simplicity. It supports multiple programming paradigms‚Äîincluding procedural, object-oriented, and functional programming‚Äîand boasts a vast ecosystem of libraries for web development, data analysis, machine learning, scientific computing, automation, and more.

print("Welcome to Python!")

2. Installation & Setup

Download Python from python.org or install via your system package manager. Then choose an editor or IDE (e.g. VS Code, PyCharm), and verify installation:

# Windows
python --version

# macOS/Linux
python3 --version

3. Variables & Data Types

Variables in Python are dynamically typed: you assign a value and the interpreter infers its type. Common built-in types include int, float, str, bool, list, dict, and more.

name = "Tanvir"
age = 25
height = 5.9
is_student = True

print(name, type(name))
print(age, type(age))
print(height, type(height))
print(is_student, type(is_student))

4. Input / Output

Use input() to read a string from the user and print() to display output. You can format strings with f-strings or concatenation.

user = input("Enter name: ")
print("Hello, " + user + "!")
print(f"Welcome, {user}!")

5. Operators

Python supports:

  • Arithmetic: +, -, *, /, %, //, **
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: and, or, not
  • Assignment: =, +=, -=, etc.
x = 10
y = 3

print(x + y, x * y)      
print(x / y, x // y)     
print(x ** y)

6. Conditional Statements

Use if, elif, and else to branch your code based on conditions.

age = 18

if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

7. Loops

for loops iterate over sequences or ranges. while loops repeat while a condition holds.

print("for loop:")
for i in range(3):
    print(i)

print("while loop:")
n = 0
while n < 3:
    print(n)
    n += 1

8. Functions

Define reusable blocks of code with def and call them by name. You can also return values.

def greet(name):
    return f"Hello, {name}!"

print(greet("Tanvir"))

9. Lists & Tuples

Lists are mutable sequences; tuples are immutable. Use square brackets for lists and parentheses for tuples.

my_list = [1, 2, 3]
my_list.append(4)

my_tuple = (4, 5, 6)
# my_tuple[0] = 7  # Error!

print(my_list, my_tuple)

10. Dictionaries & Sets

Dictionaries map keys to values. Sets store unique values and support operations like union and intersection.

my_dict = {"name": "Tanvir", "age": 25}
my_set = {1, 2, 2, 3}

print(my_dict["name"])
print(my_set)

11. String Manipulation

Strings are immutable sequences. You can concatenate, slice, and use methods like .upper(), .split(), and .replace().

text = "Data Science"
print(text.upper())
print(text.replace("Data","Big Data"))
print(text.split())

12. List Comprehension

Compact syntax for creating lists: [expr for item in iterable if condition]. It‚Äôs often faster and more readable than loops.

nums = [1, 2, 3, 4]
squares = [x**2 for x in nums if x % 2 == 0]
print(squares)

13. Error Handling

Use try/except to catch exceptions, else to run when no error, and finally for cleanup.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
else:
    print("Success")
finally:
    print("Done")

14. File Handling

Use with open(...) to read/write files safely without needing to close them explicitly.

with open("demo.txt","w") as f:
    f.write("Hello, File!")

with open("demo.txt","r") as f:
    print(f.read())

15. Modules & Packages

Organize code into modules (.py) and packages (folders with __init__.py). Import with import or from ‚Ä¶ import ‚Ä¶.

# math_utils.py
def add(a, b):
    return a + b

# main.py
from math_utils import add
print(add(3, 4))

16. Standard Libraries

Python‚Äôs standard library includes modules for date/time (datetime), randomness (random), OS operations (os), and more.

import datetime, random, os

print(datetime.datetime.now())
print(random.choice([10,20,30]))
print(os.getcwd())

17. OOP Basics (Classes & Objects)

Define classes to encapsulate data and behavior. Use __init__ to initialize attributes and self to reference the instance.

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def greet(self):
        return f"Hi, I'm {self.name}"

s = Student("Tanvir", 25)
print(s.greet())

18. Inheritance & Polymorphism

Create a subclass that inherits from a parent class and override methods for polymorphic behavior.

class Animal:
    def speak(self):
        return "..."

class Dog(Animal):
    def speak(self):
        return "Woof!"

d = Dog()
print(d.speak())

19. Lambda, map, filter, reduce

Use lambda for short anonymous functions, map() to apply to each item, filter() to select items, and functools.reduce() to aggregate.

from functools import reduce

nums = [1,2,3,4]
doubled = list(map(lambda x: x*2, nums))
evens  = list(filter(lambda x: x%2==0, nums))
sum_all = reduce(lambda a,b: a+b, nums)

print(doubled)
print(evens)
print(sum_all)

20. Iterators & Generators

Generators yield values one at a time, saving memory. Use yield in a function to make it a generator.

def count_up(n):
    for i in range(n):
        yield i

for num in count_up(5):
    print(num)

21. Decorators

Decorators wrap a function to extend behavior without modifying its code. Use @decorator syntax above the function definition.

def timer(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print("Elapsed:", time.time()-start)
        return result
    return wrapper

@timer
def compute():
    sum(range(1000000))

compute()

22. Working with JSON

Serialize Python objects to JSON with json.dumps() and deserialize with json.loads().

import json

data = {"name": "Tanvir", "age": 25}
json_str = json.dumps(data)
loaded   = json.loads(json_str)

print(json_str)
print(loaded["name"])

23. Virtual Environments

Isolate project dependencies using venv. Create and activate environments to keep requirements separate.

# Create
python -m venv env

# Activate (Windows)
env\Scripts\activate

# Activate (macOS/Linux)
source env/bin/activate

24. Unit Testing with pytest

Write tests in files named test_*.py and run pytest to automatically discover and run them.

# test_math.py
def add(a, b):
    return a + b

def test_add():
    assert add(2, 3) == 5

25. Debugging Techniques

Use pdb or IDE debuggers to set breakpoints and step through code. Simple print() statements can also help trace issues.

import pdb

def buggy():
    a = 1
    pdb.set_trace()  # Execution will pause here
    b = 0
    return a / b

buggy()

26. Introduction to NumPy & Pandas

NumPy provides fast N-dimensional arrays; Pandas builds on it to offer DataFrame and Series for tabular data analysis.

import numpy as np
import pandas as pd

arr = np.arange(10).reshape(2,5)
df  = pd.DataFrame(arr, columns=list("ABCDE"))
print(df)

27. Web Scraping

Use requests to fetch pages and BeautifulSoup to parse HTML and extract data.

import requests
from bs4 import BeautifulSoup

resp = requests.get("https://example.com")
soup = BeautifulSoup(resp.text, "html.parser")
print(soup.title.string)

28. CLI with argparse

Build command-line interfaces with argparse, automatically generating help messages.

import argparse

parser = argparse.ArgumentParser(description="Greet user")
parser.add_argument("--name", required=True, help="Your name")
args = parser.parse_args()

print(f"Hello, {args.name}!")

29. Flask Basics

Flask is a micro web framework. Define routes with @app.route and return responses or templates.

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flask!"

if __name__ == "__main__":
    app.run(debug=True)
