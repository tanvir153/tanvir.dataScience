1. Introduction

Deep Learning is a subset of machine learning that builds and trains artificial neural networks with many layers to model complex, hierarchical patterns in data. Unlike traditional algorithms, deep networks automatically learn feature representations.

2. Perceptron

The perceptron is the simplest neural unit, acting as a linear binary classifier. It computes a weighted sum of inputs, applies a step activation, and outputs 0 or 1.

def perceptron(x, w, b=0):
    z = sum(xi*wi for xi,wi in zip(x,w)) + b
    return 1 if z > 0 else 0

# Example
print(perceptron([1,0,1],[0.5,0.2,-0.3], b=0.1))  # Output: 1

3. Activations

Activation functions introduce non-linearity. Common choices:

  • Sigmoid: 1/(1+e^-x) — outputs (0,1)
  • ReLU: max(0,x) — simple and effective
  • Tanh: (e^x - e^-x)/(e^x + e^-x) — outputs (-1,1)
import math

def sigmoid(x):
    return 1 / (1 + math.exp(-x))

print(sigmoid(2))  # 0.8808

4. Forward Propagation

Computes layer outputs given inputs, weights, biases, and activation.

def forward(x, W, b, act):
    # x: input vector, W: list of weight rows, b: bias vector
    z = [sum(xi * w for xi, w in zip(x, row)) + bi for row, bi in zip(W, b)]
    return [act(zi) for zi in z]

# Example with one neuron
print(forward([1,2], [[0.5, -0.6]], [0.1], sigmoid))

5. Backpropagation

Backpropagation uses the chain rule to compute gradients of the loss w.r.t. each weight. These gradients guide weight updates via an optimizer.

See accompanying math notes for derivation!

6. Loss & Optimizers

Loss functions measure prediction error. Examples: MSE for regression, cross-entropy for classification.

import numpy as np

def mse(y, y_pred):
    return np.mean((y - y_pred) ** 2)

print(mse(np.array([1,2]), np.array([1.1,1.9])))  # 0.005

Optimizers like SGD, Adam, RMSprop adjust weights using gradients. They include learning rate schedules and momentum.

7. Layers API

Keras layers encapsulate weights, biases, and activations.

from tensorflow.keras import layers

# Dense layer with 10 units and ReLU activation
dense = layers.Dense(10, activation='relu')
print(dense)

8. Keras Basics

Keras Sequential API stacks layers in order.

from tensorflow import keras

model = keras.Sequential([
    layers.Dense(64, activation='relu', input_shape=(784,)),
    layers.Dense(10, activation='softmax')
])
print(model.summary())

9. Compile

Configure loss, optimizer, and metrics before training.

model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

10. Training

Fit the model on data for a number of epochs.

history = model.fit(

Assess model performance on test data.

loss, acc = model.evaluate(X_test, y_test)

12. CNNs

Convolutional Neural Networks for image data: conv + pooling layers. Example: build a simple CNN for MNIST.

model = keras.Sequential([
    layers.Reshape((28,28,1), input_shape=(784,)),
    layers.Conv2D(32, (3,3), activation='relu'),
    layers.MaxPooling2D(),
    layers.Flatten(),
    layers.Dense(10, activation='softmax')
])
print(model.summary())

13. RNNs

Recurrent Neural Networks for sequence data. Example: SimpleRNN.

model = keras.Sequential([
    layers.SimpleRNN(64, return_sequences=True, input_shape=(None, 10)),
    layers.SimpleRNN(32),
    layers.Dense(1)
])
print(model.summary())

18. Hyperparameter Tuning

Use KerasTuner for automated search.

import kerastuner as kt

def build_model(hp):
    model = keras.Sequential()
    for i in range(hp.Int('layers', 1, 3)):
        model.add(layers.Dense(
            units=hp.Int('units_' + str(i), 32, 128, step=32),
            activation='relu'
        ))
    model.add(layers.Dense(10, activation='softmax'))
    model.compile(
        optimizer='adam',
        loss='sparse_categorical_crossentropy',
        metrics=['accuracy']
    )
    return model

 tuner = kt.RandomSearch(
    build_model,
    objective='val_accuracy',
    max_trials=5
)
