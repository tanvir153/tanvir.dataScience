Machine Learning

Build, train, and evaluate ML models using Python and scikit-learn.

1. What is Machine Learning?

Machine learning is a subset of artificial intelligence that allows systems to learn patterns from data and make predictions or decisions without being explicitly programmed.

2. Supervised Learning: Classification Example

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# Load dataset
iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.2, random_state=42
)

# Train model
clf = RandomForestClassifier()
clf.fit(X_train, y_train)

# Predict
y_pred = clf.predict(X_test)
print("Accuracy:", accuracy_score(y_test, y_pred))

3. Evaluation Metrics

  • Accuracy: Percentage of correct predictions
  • Precision: TP / (TP + FP)
  • Recall: TP / (TP + FN)
  • F1-Score: Harmonic mean of precision and recall
