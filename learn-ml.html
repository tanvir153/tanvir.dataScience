1. Introduction

Machine Learning lets computers learn patterns and make predictions from data without being explicitly programmed. We‚Äôll cover core algorithms, data preparation, and evaluation metrics to build robust models.

2. Supervised Learning

In supervised learning, we train on labeled data (features ‚Üí target). Tasks include regression (continuous target) and classification (discrete labels).

3. Linear Regression

Fits a line y=wx+b minimizing MSE. Closed-form solution via normal equation or iterative gradient descent. 

import numpy as np

# Normal equation
X = np.array([[1,1],[1,2],[1,3]])
y = np.array([1,2,3])
w, b = np.linalg.lstsq(X, y, rcond=None)[0]
print(w, b)

4. Logistic Regression

Classification via sigmoid on a linear model. Train by maximizing log-likelihood. 

from sklearn.linear_model import LogisticRegression
clf = LogisticRegression()
# clf.fit(X_train, y_train)
# print(clf.predict(X_test))

5. Decision Trees

Tree-structured model splitting on features to minimize impurity (Gini/entropy). 

from sklearn.tree import DecisionTreeClassifier
clf = DecisionTreeClassifier(max_depth=3)

6. Support Vector Machines

Finds hyperplane maximizing margin. Kernel trick allows non-linear decision boundaries. 

from sklearn.svm import SVC
clf = SVC(kernel='rbf', C=1.0)

7. K-Nearest Neighbors

Classifies a point by majority vote among its k nearest neighbors under a distance metric. 

from sklearn.neighbors import KNeighborsClassifier
clf = KNeighborsClassifier(n_neighbors=5)

8. Ensemble Methods

Combine multiple models to improve stability and accuracy (bagging, boosting, stacking). 

from sklearn.ensemble import RandomForestClassifier
clf = RandomForestClassifier(n_estimators=100)

9. Na√Øve Bayes

Probabilistic classifier assuming feature independence. Fast and works well on text. 

from sklearn.naive_bayes import GaussianNB
clf = GaussianNB()

10. Clustering

Unsupervised grouping of points. K-Means partitions into k clusters by minimizing inertia. 

from sklearn.cluster import KMeans
km = KMeans(n_clusters=3)

11. Dimensionality Reduction

Reduce features while preserving variance or structure. PCA is the most common technique. 

from sklearn.decomposition import PCA
pca = PCA(n_components=2)

12. Feature Engineering

Create new features (polynomial, interaction terms, binning) to improve model power.

13. Pipelines

Chain preprocessing and model steps to avoid data leakage and simplify workflows. 

from sklearn.pipeline import Pipeline
pipe = Pipeline([
  ('scaler', StandardScaler()),
  ('clf', RandomForestClassifier())
])

14. Model Evaluation

Use metrics like accuracy, precision, recall, F1, ROC-AUC to judge classification performance.

15. Cross-Validation

Evaluate model stability via k-fold splits or stratified sampling. 

from sklearn.model_selection import cross_val_score
scores = cross_val_score(clf, X, y, cv=5)
print(scores.mean())

16. Hyperparameter Tuning

Search best params via GridSearchCV or RandomizedSearchCV. 

from sklearn.model_selection import GridSearchCV
param_grid = {'n_estimators': [50,100,200]}
grid = GridSearchCV(RandomForestClassifier(), param_grid, cv=3)

17. Scaling & Normalization

Standardize features to zero mean unit variance or scale to [0,1]. 

from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
