BurnoutGuard

An AI-powered journaling and forecasting tool to detect burnout risk

Project Overview

BurnoutGuard is an intelligent mental health support tool that detects early signs of emotional exhaustion, stress, and burnout using a combination of sentiment analysis, journaling, and time-series prediction. It‚Äôs designed for students, employees, and researchers to track their mental wellbeing over time.

Technologies Used

  • GPT-3.5 based journal sentiment analysis
  • Emotion classification using DistilBERT
  • Time series forecasting (ARIMA + LSTM)
  • Flask backend with REST API
  • Matplotlib and Plotly for visualization

The system extracts mood/emotion metrics from text and predicts burnout trends over time.
