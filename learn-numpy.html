1. Introduction

NumPy is the foundation of numerical computing in Python. By providing a fast, N-dimensional ndarray, it enables vectorized operations, which leverage optimized C loops under the hood for speed-ups of 10‚Äì100√ó compared to pure-Python lists. Understanding NumPy unlocks efficient data processing, scientific computing, and machine learning workflows.

2. Installation

Install via pip or conda. Always keep NumPy up-to-date for performance patches:

pip install --upgrade numpy

3. ndarray

An ndarray stores homogeneous items in contiguous memory. Key attributes:

  • shape: dimensions (rows, cols, ‚Ä¶)
  • dtype: data type (e.g., int64, float32)
  • strides: byte-steps to traverse each dimension

import numpy as np
arr = np.array([1, 2, 3])
print("arr:", arr)
print("shape:", arr.shape)
print("dtype:", arr.dtype)

4. Array Creation

NumPy offers factory functions to create arrays rapidly without loops:

  • zeros: all zeros
  • ones: all ones
  • arange: like range()
  • linspace: evenly spaced values
  • eye: identity matrix

z = np.zeros((2, 3))      # 2√ó3 zeros
r = np.arange(0, 10, 2)    # [0,2,4,6,8]
l = np.linspace(0, 1, 5)   # [0.,0.25,0.5,0.75,1.]
i = np.eye(3)              # 3√ó3 identity
print(z, r, l, i)

5. Data Types

Control storage size and precision with dtype. Conversions use views or copies: 

f32 = np.array([1.1, 2.2], dtype=np.float32)
i32 = f32.astype(np.int32)
print(f32.dtype, i32.dtype)

6. Indexing

Access elements by integer, slice, or boolean array. Indexing returns views, not copies: 

arr = np.array([10,20,30,40])
print(arr[1])       # 20
print(arr[[0,2]])   # [10,30]
mask = arr > 25
print(arr[mask])    # [30,40]

7. Slicing

Slice arrays like Python lists but returns a view into the original data. Modifying a slice changes the parent array. 

arr = np.arange(10)
slice = arr[2:8:2]
print(slice)      # [2 4 6]
slice[:] = 99
print(arr)        # slice idxs set to 99

8. Basic Math

Vectorized arithmetic applies operations elementwise without Python loops: 

a = np.array([1,2])
b = np.array([3,4])
print(a + b)   # [4,6]
print(a * b)   # [3,8]
print(a / b)   # [0.3333,0.5]

9. Universal Functions

Fast C-implemented functions for array-wide math: 

print(np.sqrt([1,4,9]))   # [1,2,3]
print(np.exp([0,1]))       # [1,2.718]

10. Broadcasting

Broadcasting stretches arrays to compatible shapes for elementwise ops without data copies. It's the engine behind concise tensor math. 

a = np.array([[1,2],[3,4]])
v = np.array([10,20])
# v broadcast to [[10,20],[10,20]]:
print(a + v)
# [[11,22],[13,24]]

11. Shape Manipulation

Reshape, transpose, flatten arrays to prepare data or adjust dims:

a = np.arange(6)
b = a.reshape(2,3).T.flatten()
print(b)  # [0,3,1,4,2,5]

12. Concatenate & Split

Join multiple arrays or split into sub-arrays:

a = np.array([1,2])
b = np.array([3,4])
print(np.concatenate([a,b]))
print(np.split(np.arange(6),3))

13. Random Module

Generate random samples for simulations & testing:

print(np.random.rand(2,2))         # uniform [0,1)
print(np.random.randint(0,10,5))    # ints 0‚Äì9

14. Linear Algebra

Use optimized routines for matrix math:

a = np.array([[1,2],[3,4]])
print(a.dot(a))                 # matrix multiply
print(np.linalg.det(a))         # determinant
print(np.linalg.inv(a))         # inverse

15. Statistics

Compute summary statistics quickly:

a = np.array([1,2,3,4])
print(a.mean(), a.std(), a.sum())  # 2.5 1.1180 10

16. Boolean Masking

Filter elements with boolean conditions:

a = np.arange(10)
print(a[a % 2 == 0])  # even numbers

17. Iteration

Efficient element-wise iteration with np.nditer:

a = np.array([[1,2],[3,4]])
for x in np.nditer(a): print(x, end=' ')
# 1 2 3 4

18. Performance Tips

Always prefer vectorized NumPy functions over Python loops. Benchmark with IPython: 

# Pure Python
 t0 = %timeit -o sum([i*i for i in range(1_000_000)])
 # NumPy vectorized
 arr = np.arange(1_000_000)
 t1 = %timeit -o arr.sum()
 print(f"Speedup: {t0.average/t1.average:.1f}√ó")

19. I/O with Files

Save/load text and binary for interoperability:

a = np.arange(5)
 np.savetxt('data.txt', a)
 b = np.loadtxt('data.txt')
 print(b)
