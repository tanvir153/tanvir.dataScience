Dropout Risk Predictor

Predict and visualize student dropout risk using machine learning

Project Overview

This project aims to identify students at risk of dropping out based on their academic and behavioral patterns. We built a Random Forest Classifier trained on attendance, subject scores, and engagement metrics.

Dataset

The dataset contains the following features:

  • Student ID
  • Attendance percentage
  • Scores in 4 recent subjects
  • Engagement level (optional)
  • Dropout label (0 = Stay, 1 = Dropout)

Model

We used RandomForestClassifier due to its robustness and ability to handle feature importance. Hyperparameter tuning was done using GridSearchCV for max_depth and n_estimators.

from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, max_depth=5)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
