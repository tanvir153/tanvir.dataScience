1. Introduction

Data visualization transforms raw numbers into intuitive graphical representations, enabling insight discovery and storytelling. We‚Äôll start with Matplotlib‚Äôs low-level API, then use Seaborn‚Äôs high-level plots and finish with interactive Plotly.

2. Pyplot API

matplotlib.pyplot provides MATLAB-style functions for creating figures and plots with simple commands.

import matplotlib.pyplot as plt
plt.plot([1,2,3],[4,5,6])
plt.show()

3. Figure & Axes

Create fine-tuned layouts by explicitly managing Figure and Axes objects.

fig, ax = plt.subplots(figsize=(6,4))
ax.plot(x, y)
ax.set_title('Title')
plt.show()

4. Styles & Themes

Use built-in styles for quick aesthetic changes:

plt.style.use('seaborn-darkgrid')
plt.plot(x, y)

5. Line Plots

Plot continuous trends with markers and custom lines:

ax.plot(x, y, marker='o', linestyle='--', color='C0')

6. Scatter Plots

Visualize point distributions with size and color scales:

ax.scatter(x, y, s=50, c=z, cmap='viridis')

7. Bar Charts

Compare categorical data:

ax.bar(categories, values, color='C1')

8. Histograms

Plot distributions of numerical data:

ax.hist(data, bins=20, alpha=0.7)

9. Box Plots

Visualize quartiles and outliers:

ax.boxplot([group1, group2])

10. Pie Charts

Show proportions with labels:

ax.pie(sizes, labels=labels, autopct='%1.1f%%')

11. Subplots

Create grids of plots for comparison:

fig, axs = plt.subplots(2,2, figsize=(8,6))

12. Customizations

Add titles, labels, ticks, grid:

ax.set_xlabel('X axis')
ax.set_ylabel('Y axis')
ax.grid(True)

13. Legends & Annotations

Add legends and annotate points:

ax.legend(['Series A'])
ax.annotate('Max', xy=(x_max, y_max), xytext=(x_max+1, y_max+1),
            arrowprops=dict(arrowstyle='->'))

14. 3D Plots

Use Matplotlib‚Äôs mplot3d toolkit:

from mpl_toolkits.mplot3d import Axes3D
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.scatter(x, y, z)

15. Seaborn Intro

High-level statistical plots built on Matplotlib.

import seaborn as sns
sns.set_style('darkgrid')

16. Seaborn Plots

Examples:

sns.histplot(data)
sns.scatterplot(x='col1', y='col2', data=df)
sns.heatmap(df.corr(), annot=True)

17. Plotly Basics

Interactive charts with Plotly Express:

import plotly.express as px
fig = px.scatter(df, x='col1', y='col2')
fig.show()
